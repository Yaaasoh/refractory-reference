# 安全確認チェックリスト

以下を確認してから作業を開始してください。

## 1. 作業ディレクトリ確認

```bash
pwd
```

- [ ] `work/` 配下で作業していますか？
- [ ] パッケージディレクトリ直下ではありませんか？

## 2. Git状態確認

```bash
git status
git diff --stat
```

- [ ] 未コミットの変更はありますか？
- [ ] 意図しないファイルが変更されていませんか？

## 3. 破壊的操作の確認

以下のコマンドは禁止されています：
- `rm -rf`
- `git clean -fd`
- `git reset --hard`

## 確認完了

すべての確認が完了したら、作業を開始してください。
