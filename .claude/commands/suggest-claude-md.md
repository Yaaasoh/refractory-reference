# CLAUDE.md更新提案

## 分析対象

このセッションで発見した以下の情報を分析してください:

1. **新しいコマンド/ツールの使用法**
   - よく使うコマンドパターン
   - 便利なオプション

2. **プロジェクト固有のパターン**
   - ファイル命名規則
   - ディレクトリ構成の発見

3. **エラーと解決策**
   - 発生したエラー
   - 解決方法

4. **ワークフローの改善点**
   - 効率化できた手順
   - 避けるべきパターン

## 提案フォーマット

以下の形式で提案を出力:

```markdown
## CLAUDE.md 更新提案

### 追加候補

#### [カテゴリ]

追加内容:
```
[追加すべきMarkdown]
```

理由: [なぜこの情報が有用か]

---

### 更新候補

#### [既存セクション名]

現在の内容:
```
[現在のCLAUDE.mdの該当部分]
```

提案する内容:
```
[更新後の内容]
```

理由: [なぜ更新が必要か]
```

## 注意事項

- **提案のみ**行い、自動更新はしない
- ユーザーの承認を得てから更新
- 肥大化防止のため、本当に重要な情報のみ提案
- CLAUDE.mdは500行以下を維持

まずCLAUDE.mdを読み込み、現在の内容を把握してください。
