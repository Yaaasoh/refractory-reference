## 1. 不定形耐火物の分類

### 1.1 基本分類

| 日本語 | 英語 | 説明 |
|--------|------|------|
| 不定形耐火物 | Monolithic refractory | 成形せずに施工する耐火物の総称 |
| キャスタブル | Castable | 流し込み施工用の不定形耐火物 |
| 吹付け材 | Gunning material | 吹付け施工用材料 |
| ラミング材 | Ramming material | 突き固め施工用材料 |
| プラスチック | Plastic refractory | 可塑性のある不定形耐火物 |

### 1.2 セメント含有量による分類（ASTM C401準拠）

| 日本語 | 英語 | CaO含有量 | 特徴 |
|--------|------|----------|------|
| 一般キャスタブル | Regular castable (CC) | >2.5% | 施工体強度高い、耐火度低下 |
| 低セメントキャスタブル | Low cement castable (LCC) | 1.0-2.5% | 高炉樋用の標準 |
| 超低セメントキャスタブル | Ultra-low cement castable (ULCC) | 0.2-1.0% | 耐食性向上 |
| ノーセメントキャスタブル | No cement castable (NCC) | ≤0.2% | リン酸塩/水ガラス/シリカゾル結合 |

*分類基準: ASTM C401-12(2018) Standard Classification of Alumina and Alumina-Silicate Castable Refractories*

### 1.3 耐火物の基本分類

| 日本語 | 英語 | 説明 |
|--------|------|------|
| 定形耐火物 | Shaped refractory | 耐火煉瓦等の一定形状を持つもの |
| 不定形耐火物 | Unshaped/Monolithic refractory | 粉粒体または練り土状、特定の形を持たない |
| 耐火煉瓦 | Firebrick | 焼成された定形耐火物 |

### 1.4 結合特性

| 日本語 | 英語 | 説明 |
|--------|------|------|
| 水和性 | Hydraulic setting | アルミナセメントと水の反応で硬化 |
| 化学結合性 | Chemical bonding | リン酸塩等の化学反応で結合 |
| 焼結結合 | Sintering bond | 高温での焼結による結合 |

### 1.5 材料系統

| 日本語 | 英語 | 説明 |
|--------|------|------|
| Al₂O₃-SiC-C系 | Alumina-SiC-Carbon | アルミナ・炭化ケイ素・炭素系材料 |
| アルミナ系 | Alumina-based | アルミナを主成分とする材料 |
| アルミナセメント系 | Calcium aluminate cement based | アルミナセメントで結合する材料 |

## 2. 物性値・測定項目

### 2.1 基本物性

| 日本語 | 英語 | 単位 | 説明 |
|--------|------|------|------|
| 比重 | Specific gravity | - | 物質の密度と水の密度の比 |
| かさ密度 | Bulk density | g/cm³ | 見かけの単位体積あたり質量 |
| 圧縮強度 | Compressive strength | N/mm² | 圧縮荷重に対する強度 |
| 曲げ強さ | Bending strength | N/mm² | 曲げ荷重に対する強度 |
| 熱膨張率 | Thermal expansion | % | 温度上昇による膨張の割合 |
| 使用限界温度 | Maximum service temperature | ℃ | 材料が使用可能な上限温度 |
| 耐火度 | Refractoriness | SK/℃ | 高温での形状維持能力 |

### 2.2 乾燥性・通気性関連

| 日本語 | 英語 | 単位 | 説明 |
|--------|------|------|------|
| 乾燥性 | Drying properties | - | 水分除去の容易さ |
| 通気率 | Permeability | - | 気体の透過しやすさ |
| 耐爆裂性 | Explosion resistance | - | 急速加熱時の破裂耐性 |
| 内部圧力 | Internal pressure | Pa | 乾燥中の内部蒸気圧 |
| 見掛け気孔率 | Apparent porosity | % | 開気孔の割合 |
| 細孔径分布 | Pore size distribution | μm | 気孔サイズの分布 |

### 2.3 粒度関連

| 日本語 | 英語 | 説明 |
|--------|------|------|
| 粒度構成 | Particle size distribution | 粒子サイズの配合比率 |
| 微粉 | Fine particles | 0.075mm以下の粒子 |
| 粗粒 | Coarse particles | 1mm以上の粒子 |
| 中間粒 | Intermediate particles | 0.075-1mmの粒子 |

### 2.4 流動性関連

| 日本語 | 英語 | 単位 | 説明 |
|--------|------|------|------|
| フロー値 | Flow value | mm | JIS R 2521準拠の流動性指標 |
| タップフロー | Tap flow | mm | 振動を加えた状態でのフロー値 |
| スランプ | Slump | mm | コンクリート・モルタルの流動性指標 |
| 添加水量 | Water addition | % | 材料に加える水の量 |

### 2.3 材料特性（定性）

| 日本語 | 英語 | 説明 |
|--------|------|------|
| 流動性 | Fluidity | 材料の流れやすさ |
| 粘性 | Viscosity | 材料の粘り気 |
| チクソトロピー性 | Thixotropy | 振動を加えると流動性が出る特性 |
| 硬練り | Stiff mix | スランプが小さい、流動性の低い状態 |
| 低水分 | Low moisture | 水セメント比が低い状態 |

## 3. 材料成分

### 3.1 主要成分

| 日本語 | 英語 | 化学式 | 機能 |
|--------|------|--------|------|
| アルミナ | Alumina | Al₂O₃ | 耐火性基材 |
| 炭化ケイ素 | Silicon carbide | SiC | 耐食性・熱伝導性 |
| 炭素 | Carbon | C | 耐スラグ性 |
| 酸化カルシウム | Calcium oxide | CaO | 結合材（セメント分） |

### 3.2 副資材

| 日本語 | 英語 | 説明 |
|--------|------|------|
| 骨材 | Aggregate | コンクリート・耐火物の骨組み材料 |
| 微粉 | Fine powder | チクソトロピー性発現に寄与 |
| 離型剤 | Release agent | 型枠からの脱型を容易にする |
