    製鉄・高炉関連用語¶

    用途: 製鉄所関連文書理解・技術報告書作成 更新日: 2025-12-17

    1. 高炉プロセス¶

    1.1 高炉操業¶

    日本語 英語 説明
    出銑（しゅっせん） Tapping 高炉から溶銑を取り出す
    出滓（しゅっさい） Slag tapping スラグを取り出す
    送風 Blasting 高炉への熱風送り込み
    装入 Charging 原料を高炉に投入
    休風 Blow-down 送風停止

    1.2 生成物¶

    日本語 英語 説明
    溶銑（ようせん） Hot metal/Pig iron 高炉で生産される溶融鉄
    スラグ（滓） Slag 鉄鉱石の不純物が分離したもの
    高炉ガス Blast furnace gas 高炉から発生するガス
    ダスト Dust 粉塵

    2. 鋳床設備¶

    2.1 樋（とい）システム¶

    日本語 英語 説明
    大樋（おおとい/主樋） Main trough/Main runner 溶銑の主流路
    枝樋（えだとい） Branch runner 大樋からの分岐
    スラグ樋（滓樋） Slag runner スラグ排出用
    傾注樋（けいちゅうとい） Tilting runner 傾動式の樋
    スキンマー樋 Skimmer runner スラグ分離部の樋

    2.2 樋の構造¶

    日本語 英語 説明
    底部 Bottom 樋の底面
    側壁 Side wall 樋の側面
    堰（せき） Weir/Dam 流れを制御する構造
    スラグライン（SL） Slag line スラグと接する高さ
    メタルライン（ML） Metal line 溶銑と接する高さ

    2.3 樋のライニング¶

    日本語 英語 説明
    ゾーン施工 Zonal lining ML/SL部位別に異なる材料を施工
    ウェア材 Wear lining 溶銑/スラグに直接接する作業層
    メタル材（ML材） ML castable メタルライン用キャスタブル
    スラグ材（SL材） SL castable スラグライン用キャスタブル
    バックライニング Backup lining 背面の断熱層
    予備樋 Stand-by trough 緊急時用の予備設備
    施工厚 Lining thickness ライニングの厚さ

    2.4 附属設備¶

    日本語 英語 説明
    マッドガン Mud gun 出銑口閉塞装置
    タップホールドリル Taphole drill 出銑口開口装置
    スキンマー Skimmer スラグ分離装置
    溶銑鍋 Torpedo car 溶銑搬送容器
    天井クレーン Overhead crane 鋳床の天井走行クレーン

    3. 耐火物の損耗¶

    3.1 損耗形態¶

    日本語 英語 説明
    溶損（ようそん） Melting wear/Dissolution 高温で溶けて減耗
    侵食（しんしょく） Erosion 流れによる物理的摩耗
    化学侵食 Chemical attack スラグ等との化学反応
    スポーリング Spalling 熱衝撃による剥離
    酸化 Oxidation 酸素による劣化

    3.2 損耗要因¶

    日本語 英語 説明
    熱衝撃 Thermal shock 急激な温度変化
    機械的摩耗 Mechanical wear 物理的な摩耗
    スラグアタック Slag attack スラグによる化学侵食
    熱サイクル Thermal cycling 温度の繰り返し変動

    4. 製鉄関連化学¶

    4.1 スラグ組成¶

    成分 化学式 役割
    酸化カルシウム CaO 塩基度調整
    酸化ケイ素 SiO₂ 酸性成分
    酸化アルミニウム Al₂O₃ 中性成分
    酸化マグネシウム MgO 塩基度調整
    酸化鉄 FeO 鉄酸化物

    4.2 スラグ特性¶

    日本語 英語 説明
    塩基度 Basicity CaO/SiO₂比
    流動性 Fluidity スラグの流れやすさ
    粘度 Viscosity スラグの粘り気
    融点 Melting point 溶融温度

    5. 操業指標¶

    日本語 英語 単位 説明
    出銑量 Hot metal production t/日 日あたり生産量
    出銑回数 Number of taps 回/日 日あたり出銑回数
    樋寿命 Trough life 回（出銑回数） 補修までの出銑回数
    稼働率 Operating rate % 運転時間割合
