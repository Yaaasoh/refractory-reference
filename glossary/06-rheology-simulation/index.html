    レオロジー・シミュレーション用語¶

    用途: シミュレーション仕様理解・技術論文作成 更新日: 2025-12-17

    1. レオロジー基礎¶

    1.1 基本概念¶

    日本語 英語 記号 単位
    せん断応力 Shear stress τ Pa
    せん断速度（ずり速度） Shear rate γ̇ s⁻¹
    粘度 Viscosity η Pa·s
    降伏応力（降伏値） Yield stress τ₀, τ_y Pa
    塑性粘度 Plastic viscosity η_p Pa·s

    1.2 流体分類¶

    日本語 英語 特徴
    ニュートン流体 Newtonian fluid 粘度一定
    非ニュートン流体 Non-Newtonian fluid 粘度が変化
    ビンガム流体 Bingham fluid 降伏応力あり、その後線形
    擬塑性流体 Pseudoplastic fluid せん断で粘度低下
    ダイラタント流体 Dilatant fluid せん断で粘度上昇

    2. 構成モデル¶

    2.1 ビンガムモデル（Bingham Model）¶

    式: τ = τ₀ + η_p · γ̇

    パラメータ 意味
    τ₀ 降伏応力
    η_p 塑性粘度

    2.2 ハーシェル・バルクレイモデル（Herschel-Bulkley Model）¶

    式: τ = τ₀ + K · γ̇ⁿ

    パラメータ 意味
    τ₀ 降伏応力
    K 稠度係数（Consistency index）
    n 流動指数（Flow index）

    特徴: n < 1: 擬塑性（shear-thinning）、n > 1: ダイラタント（shear-thickening）

    2.3 その他のモデル¶

    モデル 英語 式 特徴
    べき乗則 Power law τ = K · γ̇ⁿ 降伏応力なし
    キャッソンモデル Casson model √τ = √τ₀ + √(η·γ̇) 塗料等に適用
    クロスモデル Cross model 複雑式 広範囲のせん断速度

    3. チクソトロピー¶

    3.1 時間依存性¶

    日本語 英語 説明
    チクソトロピー Thixotropy せん断で構造破壊、静置で回復
    レオペクシー Rheopexy せん断で構造形成
    構造回復 Structural recovery 静置時の粘度回復
    構造破壊 Structural breakdown せん断による粘度低下

    3.2 チクソトロピーパラメータ¶

    日本語 英語 説明
    ヒステリシス面積 Hysteresis loop area 昇降カーブの囲む面積
    回復時間 Recovery time 元の粘度に戻る時間
    崩壊時間 Breakdown time 構造破壊に要する時間

    4. 測定方法¶

    4.1 粘度測定¶

    日本語 英語 測定原理
    回転粘度計 Rotational viscometer 回転トルク測定
    共軸円筒型 Coaxial cylinder 二重円筒間のせん断
    コーンプレート型 Cone-plate 円錐と平板間のせん断
    平行平板型 Parallel plate 2枚の平板間のせん断

    4.2 流動性測定¶

    日本語 英語 規格
    フロー試験 Flow test JIS R 2521
    タップフロー試験 Tap flow test 振動付与
    スランプ試験 Slump test JIS A 1101
    Lフロー試験 L-flow test L字型器具

    4.3 スランプ試験詳細¶

    日本語 英語 説明
    アブラムスコーン Abrams cone 標準コーン（高さ30cm）、ASTM C143
    ミニコーン Mini cone 小型コーン、セメントペースト用
    スランプ Slump 型抜き後の高さ減少量
    スプレッド（広がり） Spread 崩壊後の直径
    ワーカビリティ Workability 施工性（流動性と粘性の複合指標）
    円錐形状 Conical geometry 標準的なコーン形状
    円筒形状 Cylindrical geometry 研究用の単純形状

    4.4 スランプ試験理論¶

    用語 英語 説明
    流動レジーム Flow regime 高さ/直径比による流動パターン
    潤滑近似 Lubrication approximation 薄い流体層の近似
    プラグ流れ Plug flow 剛体的に移動する領域
    降伏領域 Yielded region 降伏応力を超えて流動する領域
    未降伏領域 Unyielded region 降伏応力以下で静止する領域

    4.5 降伏応力測定法¶

    日本語 英語 説明
    ベーン法 Vane technique 羽根を回転させて降伏応力測定
    スランプ法 Slump method スランプ値から降伏応力を推定
    スプレッド法 Spread method 広がりから降伏応力を推定
    圧縮流れ法 Squeeze flow 圧縮変形から降伏応力を測定

    5. シミュレーション¶

    5.1 数値解析手法¶

    日本語 英語 略語 特徴
    有限要素法 Finite Element Method FEM 構造解析に強い
    有限体積法 Finite Volume Method FVM 流体解析に強い
    粒子法 Particle method - 自由表面に強い
    SPH法 Smoothed Particle Hydrodynamics SPH 粒子ベース流体
    MPS法 Moving Particle Semi-implicit MPS 日本発の粒子法

    5.2 解析条件¶

    日本語 英語 説明
    境界条件 Boundary condition 解析領域端の条件
    初期条件 Initial condition 解析開始時の状態
    メッシュ Mesh/Grid 計算格子
    タイムステップ Time step 時間刻み幅
    収束判定 Convergence criterion 計算収束の判定条件

    6. 数学記号・単位¶

    記号 読み 意味
    τ タウ せん断応力
    γ̇ ガンマドット せん断速度
    η イータ 粘度
    ρ ロー 密度
    Pa パスカル 圧力・応力の単位
    Pa·s パスカル秒 粘度の単位
