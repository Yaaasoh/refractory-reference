    機械・設備用語¶

    用途: 設備仕様書理解・打合せ議事録作成 更新日: 2025-12-17

    1. 混練機・ミキサー¶

    1.1 ミキサー種類¶

    日本語 英語 説明
    プラネタリーミキサー Planetary mixer 遊星運動で混練、垂直軸
    二軸ミキサー Twin-shaft mixer 2本の軸で混練
    パン型ミキサー Pan mixer 皿状容器で混練
    連続混練機 Continuous mixer 連続的に材料を投入・排出
    バッチミキサー Batch mixer 一定量ずつ混練

    1.2 関連部品¶

    日本語 英語 説明
    混練羽根 Mixing blade 材料を混ぜる羽根
    スクレーパー Scraper 側壁付着物を掻き取る
    ライナー Liner 容器内壁の保護材
    投入口 Inlet/Charging port 材料投入部
    排出口 Outlet/Discharge port 混練物排出部

    2. 搬送・供給設備¶

    2.1 コンベア¶

    日本語 英語 説明
    ベルトコンベア Belt conveyor ベルトで材料搬送
    スクリューコンベア Screw conveyor スクリューで材料搬送
    バケットエレベーター Bucket elevator バケットで垂直搬送
    振動フィーダー Vibrating feeder 振動で材料供給

    2.2 ポンプ類¶

    日本語 英語 説明
    スクイーズポンプ Squeeze pump チューブを押し潰して圧送
    モーノポンプ Mono pump 一軸偏心ねじポンプ
    ピストンポンプ Piston pump ピストンで圧送
    ダイヤフラムポンプ Diaphragm pump 膜の往復で圧送

    3. 施工設備¶

    3.1 吹付け設備¶

    日本語 英語 説明
    吹付けノズル Gunning nozzle 材料を吹き付ける先端部
    エアコンプレッサー Air compressor 圧縮空気供給
    吹付け機（ガンニングマシン） Gunning machine 吹付け施工用機械
    ホッパー Hopper 材料一時貯留

    3.2 流し込み設備¶

    日本語 英語 説明
    型枠 Formwork/Mold 材料を流し込む枠
    バイブレーター Vibrator 振動で材料を締め固め
    棒状バイブレーター Internal vibrator 材料内部に挿入して振動
    型枠バイブレーター Form vibrator 型枠に取り付けて振動
    シュート Chute 材料を流し込む樋

    3.3 計量・制御設備¶

    日本語 英語 説明
    ロードセル Load cell 重量計測センサー
    計量ホッパー Weighing hopper 計量用の一時貯留槽
    PLC Programmable Logic Controller 制御装置
    タッチパネル Touch panel 操作用表示パネル

    4. 乾燥・養生設備¶

    日本語 英語 説明
    乾燥炉 Drying furnace 成形体を乾燥
    養生シート Curing sheet 湿潤養生用カバー
    加熱器 Heater 加熱乾燥用
    送風機 Blower 乾燥促進用

    5. 製鉄所設備（高炉周辺）¶

    日本語 英語 説明
    高炉 Blast furnace 銑鉄を生産する炉
    鋳床 Cast house/Casting floor 出銑・出滓作業場
    大樋（主樋） Main trough/Main runner 溶銑の主流路
    傾注樋 Tilting runner 傾動式の樋
    スラグ樋（滓樋） Slag runner スラグ排出用樋
    溶銑鍋 Torpedo car/Hot metal ladle 溶銑搬送容器
    スキンマー Skimmer スラグ分離装置
    出銑口 Taphole 高炉の出銑孔

    6. 測定・検査機器¶

    日本語 英語 説明
    回転粘度計 Rotational viscometer 粘度測定
    フローコーン Flow cone 流動性測定
    スランプコーン Slump cone スランプ測定
    熱電対 Thermocouple 温度測定
    赤外線放射温度計 Infrared thermometer 非接触温度測定
