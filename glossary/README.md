## 用語集の使い方

### 検索方法

1. **カテゴリ別検索**: 上記5ファイルから該当分野を選択
2. **全文検索**: Grep、IDE検索機能を使用
3. **GitHub検索**: リポジトリ内検索機能を使用

### 用語定義の形式

各用語は以下の形式で記載されています:

```markdown
### 用語名（英語名）

**定義**: （簡潔な説明）

**詳細**: （必要に応じて詳細説明）

**関連用語**: （関連する他の用語）

**出典**: （規格・文献があれば記載）
```
